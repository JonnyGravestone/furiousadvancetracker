Comment utiliser FAT ?

FAT est livré sous la forme d'un fichier gba. Ce fichier est directement utilisable sur votre ordinateur avec l'aide d'un émulateur (VirtualBoy Advance par exemple).


Mais l'avantage d'une Gameboy, c'est la portabilité ! D'autant plus qu'un son émulé n'égalera jamais l'original. Pour exécuter FAT sur une vraie Gameboy Advance, il vous faudra:

  • une vraie Gameboy (scoop!)
  • une cartouche EZ-Flash, SuperCardToSD, ou autre ... par exemple ici: http://www.r4i-r4.co.uk/products/EZ-Flash-iV-GBA-Mini-Card-for-Nintendo-GBA-and-DS-Lite.html
  • une carte SD qui va bien avec la cartouche


Après avoir investi dans tout ce petit matériel, il suffira de copier le fichier fat.gba sur la carte SD: direction la GBA !


Attention ! Cette version de FAT est expérimental. Il est inutile d'investir trop d'argent dans le but de tester sur GBA. Patience !
