Description détaillée

Une séquence contient un certain nombre de blocks.

Documentation des champs

u8 blocks[NB_BLOCKS_IN_SEQUENCE]

Un tableau contenant des numéros de blocks.

u8 transpose[NB_BLOCKS_IN_SEQUENCE]

Chaque block peut être transposé: on stocke la valeur de transpose ici. On ne peut pas la stocker dans BLOCK, car cela ne permettrait qu'une seule transposition par block. Ici, un block peut être transposé avec telle valeur dans une séquence et telle autre valeur dans une autre séquence. De même, au sein de la même séquence, on peut ainsi transposer différemment chacun des blocks présent.
