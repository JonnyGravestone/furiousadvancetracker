Description détaillée

L'instrument possède divers paramètres utiles pour modifier la sonorité d'une note: chaque note est attachée à un instrument.

Documentation des champs

u8 envelope

Valeur d'enveloppe. L'enveloppe contient trois paramètres: "steptime", "envdir", "volume".

u8 loopmode

Mode de durée du son : 1=timed, 0=continuous .

u8 soundlength

Stocke la durée du son: attention, ce paramètre ne sera appliqué que si loopmode = 1.

u8 sweep

Valeur de "sweep". Le "sweep" n'est applicable que sur le channel 1.

u8 table

Numéro de table de commandes à appliquer pour chaque note assignée à cet instrument.

u8 type

Le type de l'instrument: PULSE1, PULSE2, WAVE, NOISE, SAMPLEA, SAMPLEB

u8 voiceAndBank

Ce paramètre agrége plusieurs autres paramètres dédiés aux instruments de type WAVE : "voice", "bank", "bankmode".

u8 volumeRatio

Stocke le volume pour un instrument de type WAVE.

u8 wavedutyOrPolynomialStep

Contient la valeur du paramètre "wave" pour les instruments de type PULSE OU la valeur du paramètre "polystep" pour les instruments de type NOISE
